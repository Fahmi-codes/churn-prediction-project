## Overview
This is a **Customer Churn Prediction** project built with Python, Pandas, Scikit-Learn, and Streamlit. The project predicts whether a customer will **churn (leave)** or **stay** based on their account and usage information.  

It is designed as a **portfolio-level data science project**, with:

- Data loading, cleaning, and exploratory data analysis (EDA)  
- Machine learning model training (Random Forest classifier)  
- An interactive **Streamlit web app** to make predictions  
- Full project structure ready for deployment  

---

## Features

- Clean and preprocess customer data  
- Analyze trends and churn distribution  
- Train and evaluate a machine learning model  
- Interactive Streamlit app for **real-time predictions**  
- Deployment-ready for web access
